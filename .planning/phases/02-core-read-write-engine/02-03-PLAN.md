  <files>src/backends/node.ts, src/backends/node.test.ts</files>
  <behavior>
    - readBytes reads a file and returns Uint8Array
    - readBytes rejects traversal path (../etc/passwd) with OmniReadError
    - writeBytes writes Uint8Array to file, creates parent dirs if needed
    - writeAtomic writes to temp file (.tmp suffix in same dir) then renames to target
    - writeAtomic creates parent dirs if needed
    - readStream returns a ReadableStream of Uint8Array chunks from the file
    - readBytes on non-existent file throws OmniReadError
  </behavior>
  <action>
Create `src/backends/node.ts` exporting `nodeBackend: IOBackend`:

- `readBytes(path)`: call sanitizePath(path), then `import('node:fs/promises')` dynamically (avoids top-level import that breaks browser bundle), call fs.readFile(path), return new Uint8Array(buffer).
- `writeBytes(path, data)`: call sanitizePath(path), ensure parent dir with fs.mkdir(dirname(path), { recursive: true }), write with fs.writeFile(path, data).
- `writeAtomic(path, data)`: call sanitizePath(path), compute tmpPath = path + '.omnifs.tmp', ensure parent dir, write to tmpPath, then fs.rename(tmpPath, path). Wrap in try/finally to clean up tmpPath on error.
- `readStream(path)`: call sanitizePath(path), use `import('node:fs')` to get createReadStream, convert Node stream to web ReadableStream using `Readable.toWeb()` from `node:stream/web`.

Use dynamic `import('node:fs/promises')` and `import('node:path')` to keep the module safe for bundling in non-Node environments. Cache the import promise.

Write tests using vitest with real temp files (use `os.tmpdir()` + random suffix). Clean up in afterEach.
