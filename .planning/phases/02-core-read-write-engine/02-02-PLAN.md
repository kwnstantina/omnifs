<task type="auto" tdd="true">
  <name>Task 1: Text and JSON parsers/serializers with TDD</name>
  <files>src/parsers/text.ts, src/parsers/text.test.ts, src/parsers/json.ts, src/parsers/json.test.ts</files>
  <behavior>
    Text parser:
    - parse(Uint8Array of "hello") returns "hello"
    - parse(Uint8Array with UTF-8 BOM + "hello") returns "hello" (BOM stripped)
    - parse("hello") returns "hello" (string passthrough)
    - parse(empty Uint8Array) returns ""
    - parse(Uint8Array of UTF-8 multibyte "cafe\u0301") returns correct string

    Text serializer:
    - serialize("hello") returns "hello"
    - serialize("") returns ""

    JSON parser:
    - parse(Uint8Array of '{"a":1}') returns { a: 1 }
    - parse('[1,2,3]') returns [1, 2, 3]
    - parse(Uint8Array with BOM + '{"a":1}') returns { a: 1 } (BOM stripped)
    - parse('invalid json') throws OmniReadError
    - parse('') throws OmniReadError

    JSON serializer:
    - serialize({ a: 1 }) returns '{"a":1}' (compact by default)
    - serialize({ a: 1 }, { pretty: true }) returns pretty-printed JSON with 2-space indent
    - serialize([1, 2]) returns '[1,2]'
  </behavior>
  <action>
**Text parser** (`src/parsers/text.ts`):
- Export `textParser` object with `parse(data)` method
- If data is string, return as-is
- If data is Uint8Array, decode with TextDecoder('utf-8')
- Strip UTF-8 BOM: if first 3 bytes are 0xEF, 0xBB, 0xBF, skip them before decoding. Also strip U+FEFF from decoded string start.
- Export `textSerializer` with `serialize(data)` that calls String(data)

**JSON parser** (`src/parsers/json.ts`):
- Export `jsonParser` with `parse(data)` that first converts to string via textParser.parse, then JSON.parse. Wrap JSON.parse in try/catch — on SyntaxError, throw `new OmniReadError("Invalid JSON: " + e.message, "")` with format "json".
- Export `jsonSerializer` with `serialize(data, options?)` — if options?.pretty, use JSON.stringify(data, null, 2), else JSON.stringify(data).

Write RED tests first, then GREEN implementation, then verify.

<task type="auto" tdd="true">
  <name>Task 2: CSV/TSV parser/serializer via papaparse with TDD</name>
  <files>src/parsers/csv.ts, src/parsers/csv.test.ts, package.json, pnpm-lock.yaml</files>
  <behavior>
    CSV parser:
    - parse("a,b\n1,2") returns [["a","b"],["1","2"]]
    - parse("name,age\nAlice,30") returns [["name","age"],["Alice","30"]]
    - parse(Uint8Array of "a,b\n1,2") returns [["a","b"],["1","2"]]
    - parse("a;b\n1;2", { delimiter: ';' }) returns [["a;b"],["1;2"]] with default comma (user must specify in read options)
    - parse('') returns []
    - parse("value with \"quotes\",normal") handles RFC 4180 quoted fields

    TSV parser:
    - parse("a\tb\n1\t2") returns [["a","b"],["1","2"]]
    - parse(Uint8Array of TSV data) returns array of arrays

    CSV serializer:
    - serialize([["a","b"],["1","2"]]) returns "a,b\n1,2" (or with \r\n — accept either)
    - serialize([["a","b"],["1","2"]], { delimiter: ';' }) returns "a;b\n1;2"
    - serialize([]) returns ""
  </behavior>
  <action>
First: `pnpm add papaparse` and `pnpm add -D @types/papaparse` (papaparse has separate type package).

**CSV parser** (`src/parsers/csv.ts`):
- Import papaparse (it works in all runtimes — pure JS)
- Export `csvParser` with `parse(data, options?)`:
  1. Convert Uint8Array to string via TextDecoder if needed (reuse textParser.parse for BOM stripping)
  2. Call `papaparse.parse(str, { header: false, skipEmptyLines: true, delimiter: options?.delimiter || ',' })`
  3. Return result.data as string[][]
- Export `tsvParser` with `parse(data, options?)`: same as csvParser but with delimiter: '\t'
- Export `csvSerializer` with `serialize(data, options?)`:
  1. Use `papaparse.unparse(data, { delimiter: options?.delimiter || ',' })`
  2. Return the string result

Write RED tests first, install papaparse, then GREEN implementation.
